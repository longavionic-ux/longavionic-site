TTS-1000

TTS-1000 is a specialized test instrument for TCAS equipment functionality and performance testing, featuring multiple testing modes.

Supports TCAS interrogation and aircraft scenario simulation.

Supports ATCRBS / Mode S signal generation.

Supports ADS-B In/Out testing.

TTS-1000 adopts a modular design with a user-friendly graphical interface. It can be operated via the front panel, external keyboard/mouse, or LAN remote control.

Compatible with systems such as RGS2000NG.

  • Multi-port simulation
    • Top antenna interface (4 or 1)
    • Bottom antenna interface (4)
    • Multiple antenna impedances
    • Phase-based and amplitude-based azimuth
  • Transponder pulse adjustment
    • Transponder signal simulation output
    • ATCRBS response pulse adjustment
    • Mode S response modulation
  • Pulse measurement
    • Peak Power
    • Pulse Repeat Frequency
  • Response analysis
    • ATCRBS mode response
    • Mode S response
  • Multi-target intrusion simulation
    • Dynamic/static distance simulation
    • Dynamic/static altitude simulation
    • Dynamic/static azimuth simulation
    • Dynamic target variation simulation
