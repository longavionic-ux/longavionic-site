STAR-6000

Handheld Synthesis Tester

The STAR-6000 is a handheld synthesis tester used for field functional inspection of aircraft communication, navigation and surveillance systems.

It communicates with onboard systems through antennas and provides comprehensive CNS testing capability.

The main features are as follows:

• Integration design integrating multiple CNS signal simulation and measurement functions

• Modular design supporting flexible function configuration

• Wireless testing capability improving efficiency

• Built-in Li-ion battery supporting more than 4 hours operation

• High-brightness touch screen suitable for outdoor use

• Wide temperature design, weight less than 800g, ideal for field testing

Functions

  • VOR simulator
  • ILS simulator
  • Marker beacon (MB) simulator
  • Transponder (ATCRBS / Mode S) tester
  • DME simulator and tester
  • VHF COMM tester
  • ADS-B simulator and tester
  • TCAS simulator and tester
